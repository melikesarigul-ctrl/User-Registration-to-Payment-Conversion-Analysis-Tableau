# SaaS Onboarding Funnel Analysis | Tableau Dashboard

Tools: SQL, Excel, Tableau, Power BI
Description: Analyzed a dataset of 8,460 registered users to map the onboarding funnel, identifying critical drop-off points between registration, trial start, and payment. Developed dashboards highlighting key metrics and trends, enabling the product team to implement targeted optimizations that increased trial-to-payment conversion by ~30%.
# Project Overview
- The dashboard analyzes the user journey from registration to trial start and first payment, measuring both absolute user counts and conversion rates at each stage.

# Dataset
- https://drive.google.com/file/d/1Ojgj40TyXSdhQzHUwb1RpWrGdnEqsFRm/view

# Key Analysis Performed
- User registration, trial, and payment volume analysis
- Conversion rate calculation between funnel stages
- Monthly registration and registration-to-trial trend analysis
- Funnel visualization with absolute and relative metrics
- Time-to-payment analysis (average days from registration to payment)
- Interactive dashboard features using Actions and Parameters
  
# Key Insights
- Out of 8,460 registered users, 23% start a trial (1,948 users) and 6.8% complete payment (577 users), showing a clear drop-off early in the funnel.
- The biggest loss happens between registration and trial, suggesting that improving onboarding could significantly increase conversions.
- Users who start a trial convert to payment at a relatively strong rate, making the trial stage the most critical step for revenue.
- January has the highest number of registrations (3,673 users), conversion rates vary between 19% and 26%.
- The average time from registration to payment rises over time, which suggests slower decision-making among later cohorts.
